## Overview
今回のアップデートでは、アプリケーション構築の自動化ワークフローが統合されました。これにより、プロジェクト作成からアプリケーション実装、そしてリリースまでを一貫して自動化する基盤が強化されています。引き続きAlphaテスト段階として機能拡充を進めています。

## Key Features
- **Application Build Integration**: `build-app-simple` ワークフローを追加し、簡単なWebアプリケーションの構築を自動化しました。
- **Workflow Expansion**: プロジェクト作成とリリース作成のワークフローを統合し、`YOROZU` プロジェクトのライフサイクル管理を強化しました。

## Code Improvements
- **Document Update**: READMEおよびドキュメントを更新し、新しいワークフローの手順を追記しました。
- **Asset Management**: リリース用ヘッダー画像の生成プロセスを整備しました。
